# OTee Modbus TCP Example

A simple example on how OTee vPLC can be used to control a stepper motor and receive temperature sensor data.

## Components
- Linux PC or WSL2 (using Ubuntu 24.04)
- ESP32-S3-ETH (https://www.waveshare.com/wiki/ESP32-S3-ETH)
- Stepper motor (nema xx)
- TB6600 driver motor
- Temperature and Humidity Sensor (AHT10)
